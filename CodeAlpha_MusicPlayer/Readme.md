# Music Player | CodeAlpha

A simple web-based music player built as part of **Task 4** of the **CodeAlpha Frontend Development Internship**.


## Features
- Play / Pause music
- Next and Previous track controls
- Progress bar with seek functionality
- Displays current song title
- Responsive, dark-themed UI


## Technologies
- HTML
- CSS
- JavaScript


## How to Run
1. Open `index.html` in a web browser
2. Ensure audio files are present in `assets/music/`
