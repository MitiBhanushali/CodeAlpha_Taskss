# Image Gallery | CodeAlpha

A responsive image gallery built using **HTML, CSS, and JavaScript** as part of the **CodeAlpha Frontend Development Internship**.


## Features
- Responsive image grid
- Hover effects and smooth transitions
- Lightbox view with next/previous navigation
- Keyboard controls
- Category-based image filtering


## Technologies
- HTML
- CSS
- JavaScript


## How to Run
Open `index.html` in a web browser.
